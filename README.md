## Getting Started

### Requirements

- Node.js and npm

### Getting started

* Clone the repo
* Install dependencies by running the command in the repo
```
npm install
```
* Then, you can run locally in development mode with live reload:

```
npm run dev
```
* Open http://localhost:3000 with your favorite browser to see your project.
